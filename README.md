# Ulysses

[![Build Status](https://travis-ci.org/devshawn/ulysses.svg?branch=master)](https://travis-ci.org/devshawn/ulysses)

This project is created by Ben Sixel, Sophia Mitchellette, Matt Mitchell, and Shawn Seymour.
